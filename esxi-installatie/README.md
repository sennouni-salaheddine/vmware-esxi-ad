# Opdracht 1 — VMware ESXi 8.0 installatie

**Vak:** Datacenter & Cloud · Odisee · 2025–2026
**Docent:** Thijs Dirk

## Doel

Installeren van VMware ESXi 8.0 Update 3 (hypervisor) binnen VMware Workstation Pro 17 als geneste virtualisatie op een Windows 11 host laptop.

## Uitgevoerde stappen

1. Lege VM aanmaken in Workstation met correcte hardware-specs (nested virtualization)
2. ESXi 8.0 U3b ISO koppelen (`VMware-VMvisor-Installer-8.0U3b-24280767.iso`)
3. ESXi installatie doorlopen (schijf, root-wachtwoord)
4. Netwerkinstelling en toegang tot Host Client via browser
5. Screenshots van elke stap

## Software

- VMware Workstation Pro 17
- VMware ESXi 8.0 Update 3 (via Toledo)
