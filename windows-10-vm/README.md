# Opdracht 2 — Windows 10 Pro VM in VMware Workstation

**Vak:** Datacenter & Cloud · Odisee · 2025–2026
**Docent:** Thijs Dirk

## Doel

Aanmaken van een Windows 10 Pro virtuele machine binnen VMware Workstation Pro 17: VM aanmaken, ISO koppelen, OS installeren, en VMware Tools installeren voor optimale integratie met de host.

## Uitgevoerde stappen

1. Wizard `Create a New Virtual Machine` doorlopen
2. Windows 10 Pro evaluation ISO koppelen
3. Windows-installatie doorlopen
4. VMware Tools installeren (drivers, resolutie, clipboard-sharing)
5. Basisconfiguratie van de VM

## Software

- VMware Workstation Pro 17 (trial via Teams)
- Windows 10 Pro evaluatie-ISO
