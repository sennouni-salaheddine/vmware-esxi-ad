# Opdracht 3 — Windows Server + Workstation op geneste ESXi

**Vak:** Datacenter & Cloud · Odisee · 2025–2026
**Docent:** Thijs Dirk

## Doel

Opzetten van een volledige Windows-infrastructuur binnen een **geneste** VMware ESXi omgeving:
Windows 11 host → VMware Workstation Pro 17 → ESXi 8.0 → Windows Server + Windows Workstation.

## Uitgevoerde stappen

1. **Shared folder** configureren in Workstation zodat ISOs beschikbaar zijn voor de ESXi Host Client
2. Aanmelden op **ESXi Host Client** via browser
3. Datastore browsen en Windows-ISOs uploaden
4. Windows Server VM aanmaken op ESXi (rol AD DS/domeincontroller)
5. Windows Workstation VM aanmaken op ESXi
6. Domain join en verificatie van de infrastructuur

## Software

- Windows 11 host (laptop)
- VMware Workstation Pro 17
- VMware ESXi 8.0
- Windows Server ISO
- Windows 10/11 workstation ISO
